# AgentShield Lab – Offensive & Defensive AI Security Testing Framework

AgentShield Lab is a cybersecurity research project focused on testing, detecting, and mitigating security risks in AI agent workflows. The lab simulates common AI security attack paths such as prompt injection, insecure tool use, output manipulation, data exposure, and denial-of-service style prompt abuse, then maps those behaviours into detection logic and defensive controls.

This project is designed from a Security Operations / Blue Team perspective: not only showing how attacks work in a controlled lab, but also how analysts can detect, log, triage, and mitigate them.

See the documentation in the `docs/` directory and the simulation cases in the `simulation/` directory for more details.
