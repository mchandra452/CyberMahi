# CyberMahi

Utility scripts and notes for cybersecurity training.

## CTF Codex Companion

`ctf_codex_companion.sh` offers a small set of helpers for Capture The Flag
events. It can run simple recon scans, suggest tools for a challenge, print
common payloads, validate flag formats and even recommend drink/snack pairings.
The script will automatically install any required tools like `nmap` if they are
missing.

### Usage

```bash
bash ctf_codex_companion.sh --help
```
