# Reporting

## Tools & Syntax
- **cherrytree** - `cherrytree report.ctb`
- **script** - `script -f commands.log`
- **gnome-screenshot** - `gnome-screenshot -f proof.png`
- **pandoc** - `pandoc report.md -o report.pdf`
- **markdown** - use fenced code blocks for commands.

## Tips
- Capture clear screenshots for each step.
- Log all commands to simplify report writing.
- Provide remediation advice for each finding.
